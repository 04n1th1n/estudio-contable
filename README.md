## 📋 Descripción

Sitio web profesional y responsivo que presenta los servicios de JVL Auditores Consultores LTDA con un diseño moderno, interfaz intuitiva y optimización completa para dispositivos móviles.

## ✨ Características Principales

### 🎨 Diseño
- ✅ **Responsive Design** - Funciona perfectamente en desktop, tablet y móvil
- ✅ **Diseño Moderno** - Interfaz limpia y profesional
- ✅ **Animaciones Suaves** - Transiciones y efectos visuales atractivos
- ✅ **Navbar Pegajoso** - Navegación siempre accesible

### 📱 Secciones
1. **Navegación** - Barra de navegación con enlaces suave a cada sección
2. **Hero** - Sección de bienvenida con call-to-action
3. **Acerca de Nosotros** - Valores y diferenciales principales
4. **Servicios** - Presentación de 6 servicios profesionales
5. **Por qué Elegirnos** - 6 razones para trabajar con nosotros
6. **Contacto** - Información de contacto y formulario interactivo
7. **Footer** - Enlaces rápidos y redes sociales

### 🚀 Servicios Destacados

1. **Outsourcing Contable** - Externalización de procesos contables
2. **Asesoría Financiera** - Consultoría estratégica y operativa
3. **Revisoría Fiscal y Auditoría** - Garantía de seguridad financiera
4. **Outsourcing de Nómina** - Gestión integral de nóminas
5. **Implementación de NIIF** - Normas Internacionales de Información Financiera
6. **Declaración de Renta** - Preparación de declaraciones tributarias

## 🎓 Guía de Actualización

### Cambiar Información de Contacto
Edita el archivo `index.html` en la sección contacto:
```html
<a href="mailto:nuevo@email.com">nuevo@email.com</a>
<a href="tel:+56984984538">+56 9 8498 4538</a>
```

### Agregar Nueva Sección
1. Agrega HTML en `index.html`
2. Agrega estilos CSS en `styles.css`
3. Agrega interactividad en `script.js` si es necesario

### Cambiar Colores
Edita las variables CSS en `styles.css`:
```css
:root {
    --primary-color: #nuevo-color;
    --secondary-color: #nuevo-color;
}
```
